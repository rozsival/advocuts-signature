## 👨‍💻 Development

### 💾 Installation

```bash
nvm use
yarn install
```

### ⚙️ Configuration

Create `.env` from `.env.example`

- `ASSETS_PREFIX` = path prefix for `assets` (e.g. `https://some-remote.tld/`)

### 🚨 Linters

```bash
yarn fix
```

### 📦 Build

```bash
yarn build
```
